## How to develop
```
# install brew
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

$ brew install yarn

# fork to your repo on github
$ cd ~
$ git clone git@github.com:<you_git_username>/h5-game.git
$ cd snake
$ yarn add
```
